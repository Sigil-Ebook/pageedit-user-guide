Installation

— Getting Started —

You can download the latest version of PageEdit from the official PageEdit website and install it for your OS.

Windows

PageEdit is supported on Windows 7 and later.

  • Download PageEdit for Windows.

Double-click the file to run the installer, e.g.:

  • PageEdit_0.9.5-Win_x64.exe

It’ll extract a folder called PageEdit.

Move that PageEdit folder to a location that you’ll remember. For example: C:\Program Files\PageEdit.

macOS

PageEdit is supported on macOS 10.9 (Mountain Lion) x64 bit Intel or later

  • Download PageEdit for macOS.

Unpack the TXZ file and drag the PageEdit.app file to your Applications folder, e.g.

  • PageEdit.app-0.9.5-Mac.txz

Linux

A binary version of PageEdit is not available due to the number of versions of Linux.

However, since the source code is freely available, you can build from source for your version of Linux. And due to the effort of many volunteers, there is most likely a pre-packaged version of PageEdit built for your Linux distribution available online:

  • Building PageEdit On Linux
