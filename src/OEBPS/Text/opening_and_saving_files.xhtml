Opening and Saving Files

— Load and Save Your HTML Files —

PageEdit supports opening XHTML and text files.

Save early, save often, and make backup copies!

Always close PageEdit before you open your HTML file in another program – or the other program will lock the file and prevent you from saving in PageEdit.

Opening Documents

To open a file, click the Open button , or select the menu item File→Open or its keyboard shortcut CTRL+O then choose the file you want to open:

HTML Files

  • The HTML file and any linked files and stylesheets are imported if possible.
  • If the HTML file is invalid after cleaning, PageEdit will report the error but may not load the file. You may have to fix the errors outside of PageEdit.

Text Files

  • The text file will be converted to HTML.
  • Paragraphs must be separated by blank lines to have them split into separate paragraphs.

Saving Documents

Save your HTML file by clicking the Save button , or selecting the menu item File→Save or its shortcut CTRL+S. (You should always save your document before doing major edits in case you want to undo them.)

If you edit any text in your document, an asterisk ( * ) is added to the file’s name in the titlebar.

When you save your file PageEdit will check it for certain errors and warn you if you need to fix anything.

PageEdit will not always be able to save an HMTL file that contains errors, so frequent saves will make it easier to identify if there are errors that you need to correct.

PageEdit will also clean your source code according to your preferences when saving the file.
