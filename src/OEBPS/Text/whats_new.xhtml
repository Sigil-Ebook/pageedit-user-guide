What’s New

— New Features —

The latest version of PageEdit contains bug fixes and new features.

For a detailed list of all the change made to PageEdit, see the Changelog.

Recent Major Changes in PageEdit include:

  • The ability to pass all xhtml files in the spine in Reading Order to PageEdit via opening the OPF. This allows quick file navigation via navigation menu and next and previous arrows keys.

  • An Edit vs Preview mode toggle button that will allow links among xhtmls files in the spine to be active and work (in Mode: Preview)

  • A QtMessage handler to create a debug log file controlled via a PAGEEDIT_DEBUG_LOGFILE environment setting, to simplify user bug issue feedback

All the new features, along with all features and functionality, are of course described in this PageEdit User Guide.
